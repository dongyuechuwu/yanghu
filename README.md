# 设备养护电子档案系统

本系统为小型设备养护电子档案管理平台，核心功能包括：

- 设备信息管理（新增、编辑、列表）
- 养护记录管理（增、查、改）
- 用户管理与权限分级
- 登录与认证

## 技术栈
- PHP 7+
- SQLite3
- 原生 HTML/CSS/JS

## 表结构初始化

初始化数据库：

```sh
sqlite3 db/yanghu.sqlite3 < db/init.sql
```

## 目录结构

- inc/      公共PHP模块
- pages/    主要业务页面
- assets/   前端资源
- db/       数据库存放目录

## 默认管理员
首次运行需手动添加管理员用户到 users 表。
